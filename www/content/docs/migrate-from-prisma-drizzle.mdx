Switching to **DB Studio** is extremely simple â€” in most cases you can just replace the command.

### 1. From Prisma Studio


```json title="package.json"
// [!code --]
"studio": "npx prisma studio"
// [!code ++]
"studio": "npx db-studio"
```

That's it!  
DB Studio automatically uses the same `DATABASE_URL` from your `.env` file that Prisma uses.

**Advantages over Prisma Studio:**

- Better performance (optimized data loading)
- Modern, more responsive UI
- Works with **any** PostgreSQL database â€” not just Prisma projects
- No schema.prisma dependency
- Real-time updates & better filtering/search
